Google启动网页字体在线服务
flypeng 发布时间：08-29 来源：鹏鹏编程网 浏览：2次

文章目录

    　　目前网页上使用的字体都是常见的几种，如：Lucida Grande、Verdana、Arial、Sans-Serif，中文常见的有宋体。要想用比较个性的字体就比较麻烦了，因为您不能保证客户端是否支持这种字体，常用的方法是用Photoshop把文字制作成图片。

    　　为了让网页设计更加灵活，Google从2010年5月19日启动了网页在线字体服务，并一口气发布了十八字体族。但与Monotype Web Fonts、FontShop等字体商不同，Google这次开放的字体与它其他很多服务一样，完全免费。

    　　Google目前提供的字体共有十八种，均为目前比较流行的免费字体，包括为Android而开发的Droid系列，被称为“Fonts for Scholars”的Cardo，为代码印刷而设计的Inconsolata，业余之作而不胫走红的Yanone Kaffeesatz等等，其中最能引人注目的是字体师John Fell设计的IM Fell字体。

    　　我想每个网页设计师都会觉得这是个好消息，又一个来自Google的福音吧！

    　　想使用它们吗？使用这些字体无需付费，也不必注册，只需要做简单的几步，举个例子，如果您想在您的网页中使用“Tangerine”这种字体，只需进行如下操作：

    　　1、在网页头部加入代码：
    <link rel="stylesheet" type="text/css" href="http://fonts.googleapis.com/css?family=Tangerine">


    　　2、在您的CSS样式表中加入：
    body {
    font-family: 'Tangerine', serif;
    font-size: 48px;
    }


    　　注：这是让页面所有元素的字体都变成Tangerine，如果只想局部Tangerine字体只要改变CSS选择器就可以了！
