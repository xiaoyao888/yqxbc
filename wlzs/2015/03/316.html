Win2003路由完成局域网同享上网
http://www.flypeng.com win10系统 发布时间：2015-03-28 16:56 来源：鹏鹏电脑知识网 浏览：加载中次

    文章简介：本校计算机中心机......WindowsX......计算机一台(配置......Step1.单击......Step2.选择......Step3.新建......cdrouted......routeadd......routeadd......routeadd......//(自行把校园......routeadd......TCP/IP配置......使用此方法实现宽......

    本校合计机中心计表情房共有合计机240台，已互连为局域网，停顿拜候校内资本时经由过程校园网接口，而拜候内部资本时经由过程ADSL接口。

    Windows XP和Windows 2003都自带ADSL宽带拨号法度圭表标准，这里只需使用Windows 2003的“路由和近程拜候”法度圭表标准略加设置装备摆设，便可搞掂一切。

    1、条件

    合计机一台(设置装备摆设不用很高，只需能安装Windows 2003就行)，安装有Windows2003操作系统，内插3块网卡，网卡1:毗邻内部局域网，IP:192.168.1.1，子网掩码:255.255.255.0，网关:空，DNS:空;网卡2:毗邻ADSL,IP:自动获得，DNS:自动获得;网卡3:毗邻校园网，IP:202.203.230.2，子网掩码:255.255.255.0，网关:202.203.230.1，DNS:202.203.220.2(假定校园网网段为202.203.220.0—202.203.230.0之间，DNS效劳器为202.203.220.2);

    2、效劳器设置装备摆设

    Step1.单击“开端”—“经管对象”—“路由和近程拜候”，发动设置装备摆设导游;挑选外地效劳器，单击“操作”—“设置装备摆设并启用路由和近程拜候”。单击“下一步”，挑选“自界说设置装备摆设”—“下一步”;复选“要求拨号毗邻(由分支办公室路由使用)”和“LAN路由”—“下一步”—“完成”，即可发动路由和近程拜候。

    Step2.挑选“收集接口”，单击“操作”—“新建要求拨号接口”—“下一步”—“下一步”，挑选“使用以太网上的PPP(PPPoE)毗邻”—“下一步”—“下一步”，弹出“和谈及安然办法”选项，去掉一切钩选，单击“下一步”，输出ADSL帐号和密码，“下一步”—“完成”

    Step3.新建一批措置文件route.bat，并把其快速编制增加到“开端”—“法度圭表标准”—“发动”下，编纂route.bat形式以下:

    cdroute delete 0.0.0.0

    route add 192.168.1.0 mask 255.255.240.0 192.168.1.1

    route add 202.203.220.0 mask 255.255.240.0 202.203.230.1

    route add 202.203.221.0 mask 255.255.255.0 202.203.230.1

    //(自行把校园网的IP段增加上)

    route add 202.203.230.0 mask 255.255.255.0 202.203.230.1

    3、客户机设置装备摆设

    TCP/IP设置装备摆设以下:IP:192.168.1.x,子网掩码:255.255.255.0,网关:192.168.1.1，首选DNS效劳器:外地ADSL域名效劳器IP(可向ADSL供给商查询，如昆明电信的为:202.98.160.68)，备用DNS效劳器:202.203.220.2。此处的DNS设置很是关头，有的人会误把DNS设为:192.168.1.1。

    总结

    使用此编制完成宽带同享，可节俭采办路由器的费用，几近不占用效劳器资本，且只需往效劳器上加插网卡，便可肆意扩大客户机数目或内部出口。我单元400多台合计机使用此种编制同享一条2MADSL宽带将有一年，运转很是不变，每台合计机就象在自力使用一根2MADSL在上网，同时可以快速阅读校内资本，完成网上办公，何乐而不为!
