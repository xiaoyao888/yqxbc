Linux安装与配置JDK7
http://www.yqxbc.com  发布时间：2015-08-23 10:45 来源：未知 浏览：加载中次

文章目录

    操作系统环境：CentOS 6.4 x86_64 JDK版本：jdk-1.7.0_45-fcs.x86_64

    以前在博客文章中写到过在Linux中安装JDK，再次写这篇文章算是温故知新吧。

    之前的那篇文章是在旧的博客中，后来由于某种原因重新开了这个博客，数据也没有恢复过来，所以也只能重新记录了。

    在系统中查询当前系统是否安装JDK，命令如下：

     

     
     
     
     
    1
    2
    3
     
    rpm -qa |grep jdk
     

     

     

    其他版本的JDK可以执行：

     

     
     
     
     
    1
    2
    3
     
    rpm -qa |grep java
     

     

    由于已经安装了Open JDK，查询结果如下

    如果已经安装了Open JDK，可以先卸载Open JDK，卸载命令如下

     

     
     
     
     
    1
    2
    3
     
    rpm -e --nodeps java-1.7.0-openjdk-1.7.0.45-2.4.3.4.el6_5.x86_64
     

     

    接下来安装JDK 7，下载的文件为jdk-7u45-linux-x64.rpm

    可到Oracle官方页面去下载，下载页面链接：http://www.oracle.com/technetwork/java/javase/downloads/jdk7-downloads-1880260.html

    安装之前，先给文件授予执行的权限，之后再进行安装，命令如下

     

     
     
     
     
    1
    2
    3
    4
     
    chmod +x jdk-7u45-linux-x64.rpm
    rpm -ivh jdk-7u45-linux-x64.rpm
     

     

    安装过程，如下图

    安装完毕，配置环境变量，编辑/etc/profile文件，在文件中找到export PATH USER LOGNAME 等内容，添加以下内容：

     

     
     
     
     
    1
    2
    3
    4
    5
    6
    7
     
    JAVA_HOME=/usr/java/jdk1.7.0_45
    JRE_HOME=/usr/java/jdk1.7.0_45/jre
    PATH=$PATH:$JAVA_HOME/bin:$JRE_HOME/bin
    CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar:$JRE_HOME/lib
    export PATH USER LOGNAME MAIL HOSTNAME HISTSIZE HISTCONTROL JAVA_HOME JRE_HOME CLASSPATH
     

     

    配置完毕，使用source /etc/profile命令，使环境变量立即生效。

    查看当前安装的JDK版本，命令：java -version

    如果下载的是jdk-7u45-linux-x64.tar.gz文件，先对文件解压，之后在系统中配置环境变量，如下：

     

     
     
     
     
    1
    2
    3
     
    tar -zxvf jdk-7u45-linux-x64.tar.gz
     

     

     

     
     
     
     
    1
    2
    3
     
    mv jdk1.7.0_45/ /soft/
     

     

    之后配置环境变量，环境变量的配置与前面一直，如果是全局变量，可以编辑用户目录下的.bash_profile文件，并添加变量

     

     
     
     
     
    1
    2
    3
     
    # vi ./.bash_profile
     

     

     

     
     
     
     
    1
    2
    3
    4
    5
    6
    7
     
    JAVA_HOME=/soft/java/jdk1.7.0_45
    JRE_HOME=/soft/java/jdk1.7.0_45/jre
    PATH=$PATH:$HOME/bin:$JAVA_HOME/bin:$JRE_HOME/bin
    CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar:$JRE_HOME/lib
    export PATH JAVA_HOME JRE_HOME CLASSPATH
     

     

    同样，source ~/.bash_profile文件使环境变量生效

    检查环境变量，echo $PATH

    使用.gz文件方式安装的JDK适用于机器上部署多个版本的JDK，并且不会影响到其他版本的正常使用。

    下面介绍OpenJDK的yum源方式安装命令，仅供大家参考

    Debian, Ubuntu

     

     
     
     
     
    1
    2
    3
     
    sudo apt-get install openjdk-7-jre
     

     

    Fedora, Oracle Linux, Red Hat Enterprise Linux

     

     
     
     
     
    1
    2
    3
     
    yum install java-1.7.0-openjdk
     

     

    如果是要开发Java程序，可安装Java-develop版本

     

     
     
     
     
    1
    2
    3
     
    yum install java-1.7.0-openjdk-devel
     

     

    至此，JDK的安装就差不多了。

    参考链接：http://openjdk.java.net/install/index.html
