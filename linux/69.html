Linux系统下如何控制CVS版本配置
yqxbc 发布时间：1970-01-17 来源：未知 浏览：21次

文章目录

    1.root登陆，/etc/services里 cvspserver　2401/tcp 存在确认

    2./etc/xinetd.d/下cvspserver新规作成（文件名可改），内容：

    service cvspserver
    {
    #port = 2401
    socket_type = stream
    protocol = tcp
    wait = no
    user = root
    passenv = PATH
    server = /usr/bin/cvs
    server_args = -f --allow-root=/var/cvs pserver
    }

    3.service xinetd restart 或 /etc/init.d/xinetd restart 命令重起服务

    4.chmod 0777 –R　/var/cvs 授权

    5.cvs init 或 cvs -d /var/cvs/ init 初始化CVS

    6.useradd username 追加用户username ,passwd username 修改用户username的密码

    7.groupadd cvs追加用户组cvs

    8.usermod-g cvsusername 修改用户username的用户组为cvs

    9.chgrp -Rcvs /var/cvs
